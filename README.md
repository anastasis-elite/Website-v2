# Anastasis Site Starter

A luxury black-and-copper Next.js starter homepage built for Vercel deployment.

## Quick start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a new GitHub repository.
2. In Vercel, click **Add New > Project**.
3. Import the GitHub repository.
4. Vercel should auto-detect Next.js.
5. Click **Deploy**.

## What to replace first

- Swap the `LOGO` placeholder in `app/page.tsx` for your actual mark or image.
- Update `hello@yourdomain.com` to your real application/contact email.
- Add your brand fonts once you choose the exact type pairing.
- Expand the homepage into sections like Programs, Method, Proof, FAQ, and Apply.
